Eyevis ( Mur d'images )

Contexte

Au sein du Centre Zonal opérationnel de crise, nous avons un mur d'images dans la salle opérationnel, Eyevis étant le serveur qui permet de diriger ce mur d'images qui est dans la salle serveur. Cette machine est directement relier au 16 écrants du mur d'images et de quelques moniteurs dispersé dans la structure.

Objectifs

  • Limiter les erreurs de configuration
  • Changement de serveur Eyevis
  • Check de touts les moniteurs

Cahier des charges

  • Le projet doit être réalisé en maximum 3 mois
  • Le projet doit comporter une bonne documentation pour le service informatique
  • La machine doit avoir les nouvelles fonctionnalité de Eyevis

Gestion de projet

Afin de m'aider dans ce changement de machine j'ai eu besoin de l'aide d'un technicien de niveau 3 de chez Leyard (Eyevis).

Solution retenue et résultats

Nous avons du changer de machine car elle devenait petit par petit obsolète, changement du logiciel sur un nouveau Plus performant et plus facile a l'utilisation de nos utilisateurs, changement de certain moniteur se faisant vieux a moyenne 92678 heures chacuns.
